## 🆘 TROUBLESHOOTING

### **Q: Install banner not showing on mobile?**
**A:** 
1. ✅ Use the **floating button** (bottom-right corner)
2. ✅ Use the **profile button** (Profile tab → Download App)
3. ✅ Try **incognito mode** (clears cache)
4. ✅ Check if **already installed** (won't show again)

### **Q: Service worker not working?**
**A:** 
- ✅ Vercel provides HTTPS automatically (required)
- ✅ Clear browser cache and try again
- ✅ Check browser console (F12) for errors
- ✅ Service worker needs HTTPS (Vercel has it!)

### **Q: iOS not showing install prompt?**
**A:** 
- ✅ iOS doesn't support auto-prompt (Safari limitation)
- ✅ Tap any install button → Get clear instructions
- ✅ Manual steps: Share → "Add to Home Screen"
- ✅ Works perfectly, just manual process

**A:** 
- ✅ All features work! (tested locally)
- ✅ Cart, wishlist, orders, admin panel - all work
- ✅ Service worker caches everything
- ✅ Works offline after first visit
